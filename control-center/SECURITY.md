# Security Policy

## Supported Versions

Control Center is currently pre-1.0. Security fixes are made on `main`.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately via `0jmhml0xk@mozmail.com`.

Include:
- Affected version/commit
- Reproduction steps or proof-of-concept
- Impact assessment
- Any suggested mitigations

We will acknowledge reports as soon as possible and coordinate remediation and disclosure.
Please do not open public issues for unpatched vulnerabilities.
